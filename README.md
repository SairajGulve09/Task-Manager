# Task-Manager

# Installation and Setup

 Install Nodejs:
	https://nodejs.org/en/

Check for Nodejs:
	node -v

Add required files:
	npm install react-router-dom

To run the Program:
	npm start

----------------------------------------------------------------------------------------------------------------------------------------------------------------

Task Management Application using React that allows users to
1) Add
2) Edit
3) Delete 
4) Mark tasks as completed

Components:
• Task List Component:
 Display:
     1) Checkbox
     2) Task Name
     3) Delete Button
     4) Edit Button
     5)Add Task Button
     
• Add Task Modal/Form Component:
 Text Input:
     1) Name
     2) Description
     3) Priority (dropdown)
    *Validation is applied*
    
• Edit Task Modal:
 Text Input:
     1) Name
     2) Description
     3) Priority 

Storage: Local Storage
